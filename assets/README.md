# Assets Folder

Esta carpeta está reservada para almacenar recursos visuales y de audio del afinador:

## Archivos requeridos

### Audio
- **success.mp3** - Sonido que se reproduce cuando una cuerda se afina correctamente en modo automático

### Iconos y otros (opcional)
- Iconos SVG para el selector de instrumento
- Imágenes de fondo (si se requieren)
- Logos personalizados
- Otros recursos gráficos

## Nota importante

**success.mp3**: Coloca aquí un archivo de sonido MP3 corto (0.5-1 segundo) que se reproducirá como retroalimentación cuando el usuario afine correctamente una cuerda. Puede ser un sonido de campana, ding, o cualquier sonido agradable de éxito.

Actualmente, todo el UI está dibujado programáticamente con Canvas para mantener la portabilidad de la aplicación.
